Program Matriks, Penyelesaian Sistem Persamaan Linear dan Interpolasi Polinom
    oleh Jon Felix Germinian, Ilham Syahid S, Chokyi Ozer

Program dapat dijalankan dengan memasukan command "java Main" pada terminal/command prompt
Saat program terdijalankan, tampilan menu akan keluar
Di menu ada beberapa pilihan

Setiap Matriks diinput dengan cara 
    Misal input matriks 3x3
    Masukan :
        1 2 3
        4 5 6
        7 8 9

Pilihan 1 : Sistem Persamaan Linear
    Untuk metode Eliminasi Gauss dan Gauss-Jordan, silahkan masukan dimensi
    m : jumlah persamaan yang ada
    n : jumlah variabel

    Untuk metode Invers dan Cramer, program hanya meminta baris (Matriks wajib persegi)

Pilihan ke 2 : Determinan
    Masukan metoda yang diinginkan lalu masukan matriks, program akan mengembalikan determinan dari input Matriks

Pilihan ke 3 : Matriks Balikan
    Program akan mengembalikan Matriks balikan dari Matriks input pengguna dengan metoda yang dipilih

Pilihan ke 4 : Matriks Kofaktor
    Program akan mengembalikan Matriks kofaktor dari Matriks input pengguna

Pilihan ke 5 : Matriks Adjoin
    Program akan mengembalikan Matriks Adjoin dari Matriks input pengguna

Pilihan ke 6 : Interpolasi
    Interpolasi polinom dengan input
        jumlah titik 
        (x0, y0)
        (x1, y1)
        ...
        (xn, yn)
    Mengeluarkan hasil perhitungan Interpolasi

Pilihan ke 7 : keluar
    Keluar dari program
